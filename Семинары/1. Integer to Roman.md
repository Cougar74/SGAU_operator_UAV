# Описание
Семь различных символов используются для обозначения чисел:

| Символ | Значение |
| ------ | -------- |
| I      | 1        |
| V      | 5        |
| X      | 10       |
| L      | 50       |
| C      | 100      |
| D      | 500      |
| M      | 1000     |

Римские цифры образуются путём сложения преобразований десятичных знаков от большего к меньшему. Преобразование десятичного знака в римскую цифру происходит по следующим правилам:
- Если значение не начинается с 4 или 9, выберите символ максимального значения, которое можно вычесть из входных данных, добавьте этот символ к результату, вычтите его значение и преобразуйте остаток в римскую цифру.
- Если значение начинается с 4 или 9, используйте  форму , представляющую собой вычитание одного символа из следующего символа.
	- 4 на 1 (`I`) меньше 5 (`V`): `IV`
	- 9 на 1 (`I`) меньше 10 (`X`): `IX`
	Используются только следующие формы: 4 (`IV`), 9 (`IX`), 40 (`XL`), 90 (`XC`), 400 (`CD`) и 900 (`CM`).
- Для представления чисел, кратных 10 , можно добавлять последовательно не более **трёх** раз только степени числа 10 (`I`, `X`, `C`, `M`). Нельзя добавлять несколько раз числа 5 (`V`), 50 (`L`) или 500 (`D`).
# Задача
Дано целое число (`1 <= num <= 3999`), преобразуйте его в римскую цифру.
## Пример 1
**Дано**: num = 3749
**Вывод**: `MMMDCCXLIX`

**Объяснение**
3000 = `MMM`, т.к. 1000 (`M`) + 1000 (`M`) + 1000 (`M`)
700 = `DCC`, т.к. 500 (`D`) + 100 (`C`) + 100 (`C`)
40 = `XL`, т.к. 40 на 10 (`X`) меньше, чем 50 (`L`)
9 = `IX`, т.к. 9 на 1 (`I`) меньше, чем 10 (`X`)
## Пример 2
**Дано**: num = 58
**Вывод**: `LVIII`

**Объяснение**
50 = `L`
8 = `VIII`
## Пример 3
**Дано**: num = 1994
**Вывод**: `MCMXCIV`

**Объяснение**
1000 = `M`
900 = `CM`
90 = `XC`
4 = `IV`
# Решение
## Решение 1
```python
ROMES = ['M', 'C', 'X', 'I']
ROMES_V = ['D', 'L', 'V']
VALUES = [1000, 100, 10, 1]

res = ''
new_num = num
for i, val in enumerate(VALUES):
  print(f'---\n#{i + 1}: {new_num} -- {val} / {ROMES[i]}')

  count_romes = new_num // val
  new_num -= count_romes * val
  print(f'{new_num} // {val} = {count_romes}')

  if i and count_romes > 5:
    if count_romes == 9:
      res += ROMES[i] + ROMES[i-1]
    else:
      res += ROMES_V[i-1] + ROMES[i] * (count_romes - 5)
  elif i and count_romes == 4:
    res += ROMES[i] + ROMES_V[i-1]
  else:
    res += ROMES[i] * count_romes
    
  print(f'\n{new_num} -- {res}')

print(f'\n***\n{num} = {res}')
```
## Решение 2
```python
ROMES = ['I', 'X', 'C', 'M']
ROMES_V = ['V', 'L', 'D'] 

res = ''
for i, v in enumerate(str(num)[::-1]):
  print(f'---\n#{i + 1}: {v} -- {res}')
  if i == 3:
      res += ROMES[i] * int(v)
      continue

  if int(v) == 9:
      res += ROMES[i+1] + ROMES[i]
      continue

  if int(v) == 4:
      res += ROMES_V[i] + ROMES[i]
      continue

  res += ROMES[i] * (int(v) % 5)
  
  if int(v) >= 5:
      res += ROMES_V[i]

res = res[::-1]
print(f'\n***\n{num} = {res}')
```
