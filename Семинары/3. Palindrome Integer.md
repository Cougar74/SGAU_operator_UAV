# Описание
Число-палиндром — это число, которое одинаково читается как слева направо, так и справа налево.
Примеры таких чисел: 5, 121, 16361 и 2112.
# Задача
Дано целое число (`-2^15 <= num.length <= 2^15`). Необходимо сказать является ли число палиндромом
## Пример 1
**Дано**: num = 232
**Вывод**: `True`
## Пример 2
**Дано**: num = -1
**Вывод**: `False`
## Пример 3
**Дано**: s = 123474321
**Вывод**: `True`
