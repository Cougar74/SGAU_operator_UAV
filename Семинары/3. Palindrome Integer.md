# Описание
Число-палиндром — это число, которое одинаково читается как слева направо, так и справа налево.
Примеры таких чисел: 5, 121, 16361 и 2112.
# Задача
Дано целое число (`-2^15 <= num.length <= 2^15`). Необходимо сказать является ли число палиндромом
## Пример 1
**Дано**: num = 232
**Вывод**: `True`
## Пример 2
**Дано**: num = -1
**Вывод**: `False`
## Пример 3
**Дано**: num = 123474321
**Вывод**: `True`
# Решение
## Решение 1
```python
def isPalindrome(num: int) -> bool:
    if num < 0:
        return False

    s = str(num)
    for i in range(len(s)):
        print(f'#{i}: {s[i]} -- {s[-i-1]}')

        if s[i] != s[-i-1]:
            return False

    return True
```
## Решение 2
```python
def isPalindrome(num: int) -> bool:
    if num < 0:
        return False

    rev = 0
    temp = num

    while temp:
        digit = temp % 10
        rev = rev * 10 + digit
        temp //= 10
        print(f'{digit} -- {rev} -- {temp}')

    return rev == num
```
## Решение 3
```python
def isPalindrome(num: int) -> bool:
  if num < 0:
      return False

  elif num < 10:
    return True
    
  else:
      s = str(num)
      ind = len(s) // 2
      if s[:ind] == s[-ind:][::-1]:
          return True
      else:
          return False
```
