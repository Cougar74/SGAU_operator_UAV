# Описание
При проверке входных данных очень часто приходится проверять целостность структуры.
Пример таких файлов: `.html`, `.xml`, `.json`.
# Задача
Дана строка `s` , которая может содержать следующие символы: `(`, `)`, `{`, `}`, `[`, `]`. Необходимо определить является ли исходная строка допустимой.
Строка считается допустимой, если:
1. Открытые скобки должны закрываться скобками того же типа.
2. Открытые скобки должны закрываться в правильном порядке.
3. Каждой закрывающей скобке должна соответствовать открывающая скобка того же типа.
## Пример 1
**Дано**: s = '()'
**Вывод**: `True`
## Пример 2
**Дано**: s = '()[]{}'
**Вывод**: `True`
## Пример 3
**Дано**: s = '(]'
**Вывод**: `False`
## Пример 4
**Дано**: s = '([)]'
**Вывод**: `False`
