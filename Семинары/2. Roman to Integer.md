# Описание
Семь различных символов используются для обозначения чисел:

| Символ | Значение |
| ------ | -------- |
| I      | 1        |
| V      | 5        |
| X      | 10       |
| L      | 50       |
| C      | 100      |
| D      | 500      |
| M      | 1000     |
Например, `2` записывается, как `II` –  две единицы, сложенные вместе.
`12`записывается как  `XII`, то есть  `X + II`.
Число `27`записывается как `XXVII`, то есть `XX + V + II`.
# Задача
Дано римское число (`1 <= s.length <= 15`). Преобразуйте его в целое число.
После преобразования целое число лежит в отрезке `[1, 3999]`.
## Пример 1
**Дано**: s = "`III`"
**Вывод**: 3
## Пример 2
**Дано**: s = "`LVIII`"
**Вывод**: 58

**Объяснение:** `L` = 50, `V`= 5, `III` = 3.
## Пример 3
**Дано**: s = "`MCMXCIV`"
**Вывод**: 1994

**Объяснение:** `M` = 1000, `CM` = 900, `XC` = 90, `IV` = 4.
